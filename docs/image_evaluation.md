# 图像评测

图像评测固定使用 `data/annotations/image_test_fixed.jsonl`，输出 OCR 相似度/CER、实体 precision/recall/F1、边界错误、来源分解、坐标映射质量、框覆盖率和错误可视化。该集合是测试集，禁止依据结果回调阈值或边距。

当前 105 张结果：OCR 文本相似度 0.9164、CER 0.0887、实体 F1 0.8592、平均框 IoU 0.4124。在 0.95 覆盖阈值下完整脱敏率为 0，239 个框仅部分覆盖，另有 65 个漏检实体。`mapping_quality=exact` 只表示目标字符均可追溯到 OCR block，不表示字形边界精确。

上述数字是修复前冻结基线，本次安全映射改进未重跑或覆盖该报告。新实现增加加权估算、墨迹投影、连通域扩展、动态边距和 strict/full-block 降级；需要另建开发集验证后，才能在新的独立测试版本上重新报告指标。

生产验收应增加人工标注的真实扫描件、聊天截图和复杂版式，并同时报告检测召回与脱敏覆盖率。对高风险文档应扩大安全边距、检查预览并人工确认。
